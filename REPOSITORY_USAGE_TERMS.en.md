This repository may be studied and referenced, but "reference" does not mean direct copying.

You may learn from the ideas and reimplement them independently, but you may not use this repository as the main basis for low-modification reuse. Whether a use exceeds reasonable reference may be assessed by factors such as:

- whether the core code structure, module split, or interface design is highly similar;
- whether the main workflow, feature arrangement, or page structure is highly similar;
- whether key wording, prompts, resource files, or test samples are substantially reused; and
- whether the changes are mainly superficial, such as variable renaming, small deletions, or style replacement.

If a third party would reasonably regard your work as clearly corresponding to this repository in its overall expression, core implementation, or main functionality, it may be treated as exceeding reasonable reference.

For convenience of evaluation, if a submitted work contains cumulative reuse, similarity, or substantial dependence of 50% or more in one or more dimensions below, it will normally be regarded as exceeding the permitted scope:

The 50% threshold is only an auxiliary standard, not the only standard. Even below that threshold, a work may still be treated as impermissible if it is overall mainly based on this repository.
